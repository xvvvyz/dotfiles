# Dotfiles

## macOS

0. `xcode-select --install`
1. `git clone https://github.com/xvvvyz/dotfiles ~/.dotfiles`
2. `cd ~/.dotfiles`
3. `./install-osx.sh`
4. `gpg --card-edit`
5. `fetch`
6. `quit`

## Debian (headless)

0. `git clone https://github.com/xvvvyz/dotfiles ~/.dotfiles`
1. `cd ~/.dotfiles`
2. `./install-debian.sh`
